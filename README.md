# Projeto CPA-20

Material de apoio para estudos e simulados da certificação CPA-20, com foco em:

- Banco consolidado de questões.
- Simulador interativo no terminal.
- Interface web para quizzes.
- Scripts de análise de desempenho.

## Estrutura atual

- `estudos/`: resumos, flashcards, cronogramas e mapas mentais já existentes.
- `web/`: interface estática para acompanhamento de estudos.
- `data/`: (novo) armazena bancos de questões em formato aberto.
- `cli/`: (novo) simulador de questões em modo texto.
- `scripts/analytics/`: (novo) templates para análises de desempenho.
- `docs/`: (novo) documentação de arquitetura e rotas de evolução.

Veja a arquitetura detalhada em [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md).
